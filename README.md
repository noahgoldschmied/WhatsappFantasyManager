# Fantasy WhatsApp Bot

MVP for an AI-powered fantasy manager integrated with Yahoo Fantasy Sports.

## Setup

1. Clone repo & install deps:
   ```bash
   npm install
   ```

2. Copy `.env.example` to `.env` and fill values.

3. Run dev server:
   ```bash
   npm run dev
   ```

## Features (MVP)
- Add/drop players (with confirmation)
- Propose trades (two-step flow)
- Show team / opponent info
- Game-time updates (push)

## Notes
- Uses Twilio WhatsApp Sandbox for demo.
- Yahoo Fantasy API integration placeholder provided.
